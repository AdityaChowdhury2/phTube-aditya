1. Discuss the scope of var, let, and const

Answer: Let's discuss about scope in var,let and const.
var: var has a global scope. It means variables that are declared as var that has a global scope. It can be accessible through all the scope.
let: let has block scope. It only be accessible inside the block where it is declared.
const: const has also block scope. It only be accessible inside the block where it is declared. The difference between let and const is that const variable can not be changed later.

2. Tell us the use cases of null and undefined

Answer: null is assigned to a variable to indicate that the value is empty. undefined is the value of a variable that is declared but not assigned.

3. What do you mean by REST API?

Answer: An API is a set of rules defined how applications can connect to and communicate with the server. A REST API is an API that conforms to the design principles of the REST, or representational state transfer architectural style.
